Tyoically languages used in this course are Hyper Text Markup Language (HTML) , Cascading Style Sheet (CSS) and JavaScript . HTML is used to develop static webpages basically it is used to display the content on the webpages and CSS is used to design or style the content on the webpage and JavaScript gives the behaviour of the webpages i.e, make responsive a webpage. we are having various frameworks for front-end like Bootstrap , react , angular etc and we are also having frameworks for back-emd like nodejs, django etc. Language can be used for back-end java , javascript , ruby , python , etc The main objective of this course is to develop a website that are responsive , well defined.
